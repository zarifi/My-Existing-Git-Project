3. Convert the binary number 111100011100111110 to the base system number indicated using the grouping method discussed in class. Explain your strategies for each conversion. [12 marks]

Qaternary:The largest number that can be represented by each bit in quaternary system number is 3.Also the number 3 can be represented by two digits in binary system number(11).so we have to divide the binary number(111100011100111110) into groups of number with two digits and convert each two digits in binary to a number in quaternary.

Octal:The largest number that can be represented by each bit in octal system number is 7.Also the number 7 can be represented by three digits in binary system number(111).so we have to divide the binary number (111100011100111110)into groups of number with three digits and convert each three digits in binary to a number in octal.

Hexadecimal:The largest number that can be represented by each bit in hexadecimal system number is 15(F).Also the number 15 can be represented by four digits in binary system number(1111).so we have to divide the binary number (111100011100111110)into groups of number with four digits and convert each four digits in binary to a number in hexadecimal.
